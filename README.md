
# BOT AUTO SWAP DI OROSWAP

Kagak usah basa basi langsung comot jangan lupa bintang nya

```bash 
git clone https://github.com/kenjisubagja/oroswap/
cd oroswap
```
edit file .env nya
```bash 
PRIVATE_KEY_1="your-mnemonic"
```
run program nya
```bash 
npm install
```
```bash 
npm start
```
